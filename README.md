# Mr-Klucks-Egghunt
The Easter bunny has stolen Mr Klucks eggs

## Development

Run locally with a static server:

```bash
python3 -m http.server 8000
```

Quest/story data lives in:

- `data/quests.json`
- `data/story_flags.json`

Validate quest wiring:

```bash
node validators/validate_quests.js
```
